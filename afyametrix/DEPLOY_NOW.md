## 🔧 Environment Variables for Vercel

Add these in Vercel Dashboard **Settings → Environment Variables**:

```
NEXT_PUBLIC_APP_NAME=AfyaMetrix
NEXT_PUBLIC_APP_URL=https://afyametrix.vercel.app
NEXT_PUBLIC_API_URL=https://api.afyametrix.health/api
NEXT_PUBLIC_MOCK_API=true
NEXT_PUBLIC_ENABLE_OFFLINE_MODE=true
NEXT_PUBLIC_ENABLE_PHOTOS=true
NEXT_PUBLIC_ENABLE_AI_RECOMMENDATIONS=false
NEXT_PUBLIC_ENABLE_HEATMAP=false
NEXT_PUBLIC_ENABLE_EXPORT=false
```
